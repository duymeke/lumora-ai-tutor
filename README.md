# Lumora

Adaptive English AI Tutor built with Next.js.

## Features
- Adaptive grammar assessment
- 18-level diagnostic questions
- CEFR-based level detection
- Personalized study path
- Interactive AI tutor
- User profile system
- Dark / Light mode
- Local storage persistence

## Learning Source
English Grammar in Use  
Raymond Murphy

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
